# 📊 Análise de Desempenho das Lojas - Alura Store

## 📖 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de analisar o desempenho de quatro lojas fictícias da rede **Alura Store**. A análise busca identificar qual das unidades apresenta menor eficiência operacional, auxiliando na tomada de decisão sobre qual loja o Senhor João deveria vender para investir em um novo empreendimento.

A partir de dados de vendas reais simulados, foram exploradas métricas importantes de negócio, como faturamento total, categorias mais vendidas, satisfação dos clientes e custo médio de frete.

A análise foi realizada utilizando **Python**, com foco em manipulação de dados e visualização para apoiar decisões estratégicas.

---

## 🎯 Objetivo da Análise

Identificar a loja com menor desempenho considerando diferentes indicadores de negócio:

* Faturamento total por loja
* Categorias de produtos mais vendidas
* Média de avaliação dos clientes
* Produtos mais e menos vendidos
* Frete médio por loja

Com base nesses dados, foi elaborada uma recomendação final sobre qual unidade apresenta menor eficiência.

## 📊 Análises Realizadas

Durante o projeto foram realizadas diversas análises exploratórias para entender o comportamento das vendas nas lojas.

### 💰 Faturamento das Lojas

Foi calculado o faturamento total de cada loja com base no preço dos produtos e na quantidade de vendas.

### 🛍️ Vendas por Categoria

Foram analisadas as categorias de produtos mais populares em cada loja, identificando quais segmentos apresentam maior demanda.

### ⭐ Avaliação dos Clientes

A média das avaliações dos clientes foi analisada para entender o nível de satisfação com as compras realizadas.

### 📦 Produtos Mais e Menos Vendidos

Foram identificados os produtos com maior volume de vendas e aqueles com menor saída.

### 🚚 Frete Médio

O custo médio de frete foi avaliado para entender o impacto logístico em cada loja.

## 📈 Visualizações

Durante a análise foram criados gráficos utilizando **Matplotlib**, incluindo:

* Gráfico de barras do faturamento por loja
* Comparação da média de avaliações
* Frete médio por loja
* Distribuição de vendas por categoria

Essas visualizações ajudaram a interpretar os dados de forma mais clara e facilitaram a comparação entre as lojas.

## 🧠 Conclusão

Após analisar os principais indicadores de desempenho das quatro lojas, foi possível identificar diferenças importantes entre elas em relação a faturamento, satisfação dos clientes e custos logísticos.

Com base nesses dados, foi possível recomendar qual unidade apresenta menor eficiência e poderia ser considerada para venda, permitindo que os investimentos sejam direcionados para lojas com melhor desempenho e maior potencial de crescimento.
